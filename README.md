# Lampa Smile Reactions

Плагин для Lampa, который добавляет три компактные реакции на постеры в лентах, категориях и поиске:

- огонь
- палец вверх
- какашка

Плагин не меняет экран полной карточки фильма или сериала.

## Установка

Добавьте в Lampa как внешний плагин:

```text
http://192.168.1.240:8080/smile_reactions.js
```

На сервере включен `Cache-Control: no-store` для `boot.js` и `smile_reactions.js`, а `boot.js` дополнительно грузит плагин с cache-busting query string. После обновления файла достаточно полностью перезапустить Lampa.

## Что внутри

- `smile_reactions.js` — основной Lampa-плагин.
- `boot.js` — серверный загрузчик плагина и локальных настроек Lampa.
- `Dockerfile.lampa` — Dockerfile-оверлей для копирования `boot.js` и плагина в Lampa image.
- `nginx.conf` — gateway-конфиг с no-cache правилами для обновляемых JS-файлов.

## Поведение

Плагин ищет стандартные постерные карточки Lampa (`.card`) и добавляет блок реакций внутрь `.card__view`, слева от штатного рейтинга `.card__vote`.

Разметка адаптивная:

- блок занимает только доступное место до рейтинга;
- размер считается от реального размера рейтинга;
- при нехватке места скрываются счетчики, но три иконки остаются видимыми;
- layout пересчитывается при изменении DOM, resize и orientation change.

## Проверка

```sh
node --check smile_reactions.js
node --check boot.js
```
